Detailed Description

template<typename Payload, typename DefaultLoader, typename... Versions>
class bl::bf::VersionedBinaryFile< Payload, DefaultLoader, Versions >

Helper class to facilitate the versioning of binary files. Versions are represented by the provided loaders. Version numbers are derived from the template ordering so care must be taken to keep it consistent. A special header value is written to versioned files in order to detect versioned from non-versioned files.

Template Parameters
PayloadThe type of object to read and write
DefaultLoaderThe VersionedPayloadLoader to use if version information is missing
VersionsA list of VersionedPayloadLoader for all the versions of the object
