Detailed Description

template<typename T>
class bl::bf::SerializableField< T >

Template class that represents a serializable field for BinaryFile use. Strings and integral types are supported out of the box, as well as SerializableObjects. User defined types must either implement SerializableObject or a custom version of Serializer.

Template Parameters
TThe type of the field to be serialized
