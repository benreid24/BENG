Detailed Description

template<typename TResourceType>
class bl::BackgroundResourceLoader< TResourceType >

Utility class for loading resources in the background while monitoring the progress. Resource loading is done from the calling thread, and monitoring is done on a second thread. This is to eliminate any context issues with loading SFML resources.
