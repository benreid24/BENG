Detailed Description

template<typename... TEvents>
class bl::MultiEventListener< TEvents >

Listener of multiple event types. Listeners should inherit this class with the types of events they would like to subscribe to as template params. An observe method from MultiEventListenerBase will be created for each passed type and must be implemented.
